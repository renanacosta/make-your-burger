## Project setup
```
npm install
```

### Install this dependency to make Vue work
```
npm i vue@3.2.26
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and hot-reloads to backend
```
npm run backend
```

### Compiles and minifies for production
```
npm run build
```
